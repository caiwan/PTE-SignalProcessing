WebGL-ről

Korábbi tapasztalataim nyomán a feladatok implementálását OpenGL ES 2.0 for web - továbbiakban webgl2 - segítségével végeztem el. Az OpenGL ES az OpenGL API beágyazott rendszerek (telefonok, tabletek, kis teljesítményű ARM eszközök) számára kialakított, és megelelően leegyszerűsített változata. Ennek segítségével lehetőség nyílik hardveresen gyorsított valós idejű grafika megjelenítésére, mely egy böngészőbe ágyazható, és programozható.
